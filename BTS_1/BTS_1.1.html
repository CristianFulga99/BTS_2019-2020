My First Bootstrap Page

1) A cosa serve bootstrap?

Botstrap è un framework che consente di costruire pagine web responsive.

2) Cosa devi importare per creare una pagina web con bootstrap?

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.2.1/css/bootstrap.min.css">

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"> </script>

<script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.6/umd/popper.min.js"> </script>

<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.2.1/js/bootstrap.min.js"> </script>


3) Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Con CDN (Content Delivery Network) gli utenti che usano bootstrap possono caricare file css, java e altri sui server si BTS

4) A cosa serve questo attributo del tag meta? (content="width=device-width, initial-scale=1)

Questo tag viene utilizzato per controllare il ridimensionamento delle pagine sui dispositivi

5)Che differenza c’è tra Container e Container Fluid?

Fino ad ora è stato creato un file in container-fluid che fornisce un contenitore che copre l'intera larghezza

Mentre questo è un container normale, che è reattivo in base al contenuto all'interno
